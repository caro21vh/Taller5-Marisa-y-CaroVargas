# Sistema de Irrigación Automático



El sistema permite mantener la irrigación de las plantas al analizar la humedad del suelo en el que se encuentran, y con base a los datos obtenidos, procede a realizar el riego cuando es necesario.



**Elementos:**

* Planta
* Maceta
* Tierra
* Sensor de humedad del suelo
* Contenedor de agua
* Electroválvula
* Arduino



**Entradas:**
Sensor de humedad del suelo: Señal analógica

Sensor de nivel de agua: Señal analógica



**Salidas:**

Electroválvula: Señal digital

Led indicador de reserva de agua: Señal digital
